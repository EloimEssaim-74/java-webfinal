## 系统规模概览

| 维度 | 指标 |
|------|------|
| 微服务数 | 8（gateway、user、article、interact、ai-assistant、tag-extract、compliance + common 库）|
| Docker 容器 | 12（含 Nginx、MySQL 主从、Redis、RabbitMQ、Nacos）|
| API 端点 | 15 个 REST 端点 |
| 数据库表 | 3 张（users、articles、comments）|
| Redis 数据结构 | 7 种（ZSET、String、Stream、List、Pub/Sub Channel）|
| RabbitMQ 队列 | 2 个持久队列、1 个 Topic Exchange |
| 代码行数 | ~8,000 行 Java + ~3,000 行前端 JS/CSS |
