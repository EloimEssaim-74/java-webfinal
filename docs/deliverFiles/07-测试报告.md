## 8. 关键链路请求/响应示例

### 8.1 注册 → 登录 → 发布文章 → 点赞 → 评论 → 热搜

```bash
# 1. 注册
curl -X POST http://localhost/api/user/register \
  -H "Content-Type: application/json" \
  -d '{"username":"demo","password":"demo123456"}'
# → {"code":200,"message":"success","data":{"id":16,"username":"demo","role":"user",...}}

# 2. 登录
curl -X POST http://localhost/api/user/login \
  -H "Content-Type: application/json" \
  -d '{"username":"demo","password":"demo123456"}'
# → {"code":200,"message":"success","data":{"token":"eyJ...","tokenType":"Bearer","userId":16,...}}

# 3. 发布文章
curl -X POST http://localhost/api/articles \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $TOKEN" \
  -d '{"title":"测试文章","content":"文章正文","status":"PUBLISHED"}'
# → {"code":200,"message":"success","data":{"id":31,"status":"PUBLISHED",...}}

# 4. 点赞
curl -X POST http://localhost/api/articles/31/like \
  -H "Authorization: Bearer $TOKEN"
# → {"code":200,"message":"success"}

# 5. 评论
curl -X POST http://localhost/api/comments \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $TOKEN" \
  -d '{"articleId":31,"content":"好文章！"}'
# → {"code":200,"message":"success"}

# 6. 查看热搜
curl http://localhost/api/trending
# → {"code":200,"message":"success","data":[{"id":31,"title":"测试文章","heatScore":4,...}]}
```
