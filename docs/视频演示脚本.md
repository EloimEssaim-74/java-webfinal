# 视频演示脚本 — Knowledge Platform

> **时长**: 4 分 30 秒（目标 3-5 分钟）
> **录制方式**: 屏幕录制（OBS / Windows自带 Xbox Game Bar）
> **演示终端**: WSL2 (左边) + 浏览器 Swagger UI (右边, 可选画中画) + VSCode (后台源码展示)
> **旁白**: 每步口播，不用提前录音，录制时同步讲

---

## 准备工作（录制开始前）

```bash
# 1. 确保 Docker 可用
docker ps | grep -c "Up"   # 应输出 14

# 2. 准备好终端窗口布局（建议三个窗口平铺）:
#    ┌──────────────┬───────────┐
#    │ 终端1: 主操作 │ 终端2: 日志│
#    │ (curl/脚本)  │ (docker   │
#    │              │  logs -f) │
#    └──────────────┴───────────┘

# 3. 提前在终端2启动消费者日志监控(录前30秒开始):
docker logs -f kb-tag-extract 2>&1 | grep --line-buffered "收到\|完成" &
docker logs -f kb-compliance 2>&1 | grep --line-buffered "收到\|检测" &

# 4. 准备浏览器打开 Swagger UI (可选画中画):
# http://localhost/api/user/swagger-ui.html
```

### 0:20-0:50 — 认证：注册→登录→安全验证

**画面**: 终端1

```bash
# 注册（演示安全加固：role=admin被强制为user）
curl -s -X POST http://localhost/api/user/register \
  -H "Content-Type: application/json" \
  -d '{"username":"video_demo","password":"demo123456","role":"admin"}' \
  | python3 -c "import sys,json; d=json.load(sys.stdin); print(f'注册: role={d[\"data\"][\"role\"]}')"

echo ""

# 登录获取Token
LOGIN=$(curl -s -X POST http://localhost/api/user/login \
  -H "Content-Type: application/json" \
  -d '{"username":"alice","password":"alice123456"}')
TOKEN=$(echo $LOGIN | python3 -c "import sys,json; print(json.load(sys.stdin)['data']['token'])")
echo "登录: Token=${TOKEN:0:20}..."
```

**口播**: 「注册一个用户——注意我传了 `role=admin`，但系统强制返回 `user`。这是代码审查中修复的安全漏洞。登录获取 JWT Token。」

### 0:50-1:30 — 核心链路：创建→发布→异步处理→查看标签

**画面**: 终端1(左) + 终端2(右,消费者日志实时滚动)

```bash
# 创建文章
ARTICLE=$(curl -s -X POST http://localhost/api/articles \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $TOKEN" \
  -d '{"title":"Kubernetes 生产级容器编排完全指南","content":"本文深入探讨 Docker 容器化与 Kubernetes 集群管理的核心实践，包括 Pod 调度策略、Service 服务发现、HPA 水平自动扩缩容、Helm Chart 打包与 CI/CD 集成。"}')
AID=$(echo $ARTICLE | python3 -c "import sys,json; print(json.load(sys.stdin)['data']['id'])")
echo "创建: id=$AID status=DRAFT"

# 发布（触发异步处理）
curl -s -X PUT "http://localhost/api/articles/$AID/publish" \
  -H "Authorization: Bearer $TOKEN" \
  | python3 -c "import sys,json; print(f'发布: status={json.load(sys.stdin)[\"data\"][\"status\"]}')"

echo "等待异步处理..."
sleep 5

# 查看标签提取和合规检测结果
curl -s "http://localhost/api/articles/$AID" | python3 -c "
import sys,json
d=json.load(sys.stdin)['data']
print(f'tags: {d[\"tags\"]}')
print(f'auditResult: {d[\"auditResult\"]}')
"
```

**关键画面**: 终端2 同步显示 `收到标签提取任务 → 标签提取完成` 和 `收到合规检测任务 → auditResult=PASS`

**口播**: 「创建文章，状态被强制为 DRAFT。发布触发 RabbitMQ 消息——右侧终端可以看到消费者实时处理日志。标签提取和合规检测在 5 秒内完成，结果写回数据库。」

## 录制技巧

| 技巧 | 说明 |
|------|------|
| **提前预热** | 录制前 30 秒执行一遍所有 curl，确保 API 响应快 |
| **字体放大** | 终端字号调到 16pt+，方便视频压缩后还能看清 |
| **安静环境** | 关闭通知、微信、邮件弹窗 |
| **一次录完** | 中间不停顿，卡壳了重新来比剪辑快 |
| **关键画面停留** | 执行完一个 curl 后停 2 秒，让观众看清输出 |
| **消费者日志并排** | 终端2 放在右边显示 MQ 日志，是最有说服力的画面 |

## 可选增强

- **Swagger UI 画中画**: 浏览器打开 `http://localhost/api/user/swagger-ui.html`，录制时切到浏览器展示可用接口列表
- **PPT 插入**: 视频开头放 3 秒 PPT 封面页，结尾放 3 秒 GitHub 链接页
- **容器仪表盘**: `docker stats --no-stream` 展示各容器资源占用
